## Definir Nomes para o podcast
Você é um roteirista de podcast e vamos criar um podcast de tecnologia focado em machine learning e eu gostaria de uma ajuda sua para criar 5 sugestões de nomes criativos para um podcast de machine learning feito por nerds, e que tenha um trocadilho nerd no nome.

O podcast vai falar sobre otimização de hiperparâmetros dos modelos de machine learning e qual a melhor forma de utilizalos.

{REGRAS}
O nome deve ser enxuto, um nome e um subtitulo
O nome tenha algum trocadilho nerd com nome de franquias conhecidas como Star Wars, gravity falls, Sobrenatural.
O nome deve conter alguma palavra forte que remeta a machine learning.

{REGRAS NEGATIVAS}
Não quero que o nome contenha palavas em ingles

## Definir a capa

A podcast host with a microphone and a mage's garb. Retro style, game style, medium resolution, down angle, game boy colors, 8 bits

## Textos

Você é um roteirista de podcast e vamos criar um roteiro de um podcast de tecnologia focado em machine learning cujo o nome é Algoritmo Natural: A Magia dos Hiperparâmetros e tem foco em machine learning com o público alvo de iniciantes em machine learning.

o formato deve ser:
[INTRODUÇÃO]
[O QUE É MACHINE LEARNING]
[O QUE SÃO HIPERPARAMETROS]
[COMO AJUSTAR OS HIPERPARAMETROS]
[COMO IDENTIFICAR QUANDO O AJUSTE DOS HIPERPARAMETROS ESTÃO BONS]
[CURIOSIDADES]
[FINALIZAÇÃO]

{REGRAS}
No bloco [INTRODUÇÃO] subistitua por uma introdução dos vídeos do canal "ei nerd", como se fosse escrito por Peter Jordan
No bloco [O QUE É MACHINE LEARNING] explique o que é machine learning de forma didatica.
No bloco [O QUE SÃO HIPERPARAMETROS] explique o que são hiperparametros dos modelos de forma didatica
No bloco [COMO AJUSTAR OS HIPERPARAMETROS] explique quais hiperparametros podemos ajustar em alguns modelos, no máximo 3 modelos de exemplo. Explique de forma didática.
No bloco [COMO IDENTIFICAR QUANDO O AJUSTE DOS HIPERPARAMETROS ESTÃO BONS] explique o que levar em considerçaão para avaliar quando os ajustes estão bons. Explique de forma didatica.
No bloco [CURIOSIDADES] substitua por uma curiosidade sobre machine learning. Explicando de forma didática.
No bloco [FINALIZAÇÃO] substitua por uma despedida cool com o final "Eu me chamo Ewerton Pinheiro e esse foi o MageCast de hoje"

Use termos de fácil explicação
O podcast vai ser apresentado somente por uma pessoa, chamada Ewerton Pinheiro
O podcast deve ser curto

{REGRAS NEGATIVAS}
Não use muitos termos técnicos
Não ultrapasse 5 minutos de duração
